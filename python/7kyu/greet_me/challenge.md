# Greet Me

## Challenge Description

Write a method that takes one argument as name and then greets that name, capitalized and ends with an exclamation point.

Example:

```
"riley" --> "Hello Riley!"
"JACK"  --> "Hello Jack!"
```

## Challenge Solutions

### Solution 01

```python
def greet(name): 
    return f"Hello {name.capitalize()}!"
```
